# The Microbial Dark Matter · A Manifesto

A Renaissance-styled essay on microbial dark matter — the bacteria living in your gut (and everywhere else) that we have never been able to grow in a lab.

**Live site:** https://miyu-horiuchi.github.io/microbe-manifesto/

## About

A single-page manifesto exploring the vast majority of microbial life that remains uncultivated and unstudied, including a focused section on the *Christensenella* genus. Built as a static HTML page with custom typography and dot-matrix figures.

## Local development

Open `index.html` directly in a browser, or serve the directory:

```sh
python3 -m http.server 8000
```

## Deployment

Deployed via GitHub Pages from the `main` branch.
